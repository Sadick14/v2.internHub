# **App Name**: InternshipTrack

## Core Features:

- Student Verification & Onboarding: Secure student portal with role-based access control, integrated with university credentials for profile setup and internship details, automatically determines student's Faculty and Department.
- Lecturer (Internal Supervisor) Assignment: Automated lecturer assignment based on internship location, workload, and departmental alignment, or manual override.
- Company Supervisor Integration: Dedicated company supervisor portal linked to assigned interns, providing access to view tasks, approve reports, and offer feedback, displaying intern's Faculty and Department.
- Daily Task Declaration: Structured task entry with fields for description, expected outcome, and learning objective, along with progress tracking.
- Multi-Stakeholder Engagement & Communication: In-app messaging for direct communication, feedback loops, and targeted alerts for pending approvals, late submissions, and important announcements, with departmental targeting capabilities.
- Daily Work Report Submission: Structured report template linked to declared tasks, multimedia attachment support, and a supervisor approval workflow visible to Heads of Department.
- AI Report Summarization Tool: Generate summary of the student work completed and learning outcomes achieved based on student inputs.

## Style Guidelines:

- Primary color: #4CAF50 (Green). This color reflects growth, learning, and a supportive environment. It's vibrant yet professional.
- Background color: #F0F2F0 (Light Gray). This provides a clean, neutral backdrop that ensures readability and reduces eye strain.
- Accent color: #2196F3 (Blue). Use this to highlight key interactive elements, such as buttons and links, ensuring they stand out.
- Body font: 'Inter' sans-serif, for body text
- Headline font: 'Space Grotesk' sans-serif, for headlines
- Consistent use of flat, modern icons to represent different functions. Icons should be intuitive and easily recognizable.
- Clean and organized layout with clear sections for different functionalities. Prioritize easy navigation and logical grouping of content.
